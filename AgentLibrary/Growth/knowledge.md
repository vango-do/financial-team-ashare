# 成长风格分析师 (Growth Analyst) - A股投资智慧库

## 核心投资哲学 (Core Philosophy)

### 1. 寻找十倍股 (Finding Tenbaggers)
- "投资于高增长公司"
- "成长是最好的价值"

### 2. 渗透率理论 (Penetration Theory)
- 渗透率<10%：概念期
- 渗透率10-50%：成长期（黄金期）
- 渗透率50-80%：成熟期
- 渗透率>80%：衰退期

### 3. 赛道论 (Sector Thesis)
- "选择比努力更重要"
- "好赛道出牛股"

## A股特色赛道 (A-Share Sectors)

### 专精特新 (Specialized & New)
- 定义：细分市场龙头+技术创新
- 筛选标准：
  - 研发费用率>5%
  - 营收增速>20%
  - 毛利率>30%
  - 隐形冠军

### 低空经济 (Low-altitude Economy)
- eVTOL、无人机配送
- 政策催化：多地出台支持政策
- 产业链：整机、零部件、运营

### 半导体 (Semiconductor)
- 国产替代主线
- 设备>设计>封测
- 关注：先进制程、国产设备

## 成长股筛选模型 (Screening Model)

### 核心指标
| 指标 | 要求 | 说明 |
|------|------|------|
| 营收增速 | >30% | 连续3年 |
| 净利润增速 | >30% | 连续2年 |
| 研发费用率 | >8% | 科技股 |
| 毛利率 | >35% | 护城河 |
| 存货周转 | 稳定 | 不积压 |

### PEG估值
```
PEG = PE / 净利润增长率
< 1 = 被低估
1-2 = 合理
> 2 = 泡沫
```

### A股成长溢价
- 科创板：PEG可容忍至2.0
- 创业板：PEG可容忍至1.5
- 主板：PEG需<1.2

## 产业链分析 (Industry Chain)

### 上中下游分析法
- **上游**：技术壁垒、原材料成本
- **中游**：产能、良率、规模
- **下游**：需求、渗透率、客户结构

### 供需缺口
- 供不应求 → 涨价 → 利润爆发
- 供过于求 → 降价 → 业绩杀跌

## 成长股风险 (Risk Factors)

### 估值风险
- 增长不及预期 → 戴维斯双杀
- 回调幅度>50%常见

### 技术风险
- 技术路线变化
- 被竞争对手超越

### 政策风险
- 集采降价
- 补贴退坡
- 出口管制

## 实战技巧 (Practical Tips)

### 买点选择
1. 回调至60日均线
2. 缩量企稳
3. 业绩公告后

### 卖点选择
- PE>100且增速放缓
- 行业景气度下行
- 竞争格局恶化
- 估值透支3年以上

### 仓位管理
- 单票不超20%
- 行业不超40%
- 成长股不超60%
