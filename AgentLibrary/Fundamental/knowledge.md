# 基本面分析师 (Fundamental Analyst) - A股投资智慧库

## 核心方法论 (Core Methodology)

### 1. 财务报表分析 (Financial Statement Analysis)
- 资产负债表：家底厚不厚
- 利润表：赚钱能力强不强
- 现金流量表：真金白银到账没

### 2. 三大表勾稽关系
```
净利润 → 经营现金流
总资产 → 负债 + 所有者权益
收入确认 → 现金流匹配
```

### 披露时间
| 报告类型 | 披露时间 | 关注重点 |
|----------|----------|----------|
| 年报 | 4月30日前 | 最全面 |

- 主板：净利润同比>50%或<50%需预告
- 创业板/科创板：强制预告

## A股特殊科目 (Special Items)

### 非经常性损益 (Non-recurring Items)
- 政府补助
- 投资收益
- 资产减值损失转回
- 一次性股权转让

**识别方法**：连续3年扣除非经常性损益后的净利润

### 商誉减值 (Goodwill Impairment)
- 收购时溢价太高
- 业绩承诺不达标
- 减值=业绩洗澡

### 股权激励 (Stock Incentives)
- 行权条件是否容易达成
- 费用摊销影响利润
- 激励是否绑定核心人员

## 估值方法 (Valuation)

### PE估值法
```
合理PE = 无风险收益率的倒数 × 增长率系数
```
- 成长股：PEG = 1
- 成熟股：PE = 10-15

### DCF估值
- 折现率：WACC
- 永续增长率：2-3%
- 敏感性分析很重要

### 分红折现
- 高股息股：股息/股价 > 4%
- 稳定分红公司适用

## 财务排雷清单 (Red Flags)

### 必须排除
- [ ] 经营现金流连续3年负
- [ ] 存货周转越来越慢
- [ ] 应收账款异常增长
- [ ] 大存大贷
- [ ] 商誉占净资产>50%
- [ ] 关联交易异常
- [ ] 频繁更换审计所
- [ ] 实际控制人变更

### 预警信号
- 毛利率突然提升（可能虚增）
- 投资收益占比>30%
- 政府补助占比>20%
